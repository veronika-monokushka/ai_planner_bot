# ИИ-помощник для питания AI-Flow

**Стэк**: Python + LangChain + Telegram Bot API  

**Статус:** В разработке

 **Описание:** AI-Flow - это ИИ-агент на базе LangChain с интерфейсом через Telegram Bot API, основной функционал:
1. Генерация планов питания с учётом уровня калорий и целей (похудение/набор/поддержание)
2. Использование сохранённых рецептов с посмощью RAG
3. Составление списка продуктов
4. Добавление и управление напоминаниями
5. Хранение истории взвешиваний
    Реализована гибридная логика: функции(инструменты) можно использовать через интерфейс бота или же через диалог с агентом.
